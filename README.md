# Comparación de Programación Tradicional y POO

Este proyecto compara dos enfoques de programación en Python para calcular el promedio semanal del clima.

## Programación Tradicional
Utiliza funciones independientes para el ingreso de datos y el cálculo del promedio.

## Programación Orientada a Objetos
Organiza el código mediante una clase que encapsula los datos y métodos relacionados al clima.

## Conclusión
Ambos enfoques resuelven el problema, pero la POO ofrece mejor organización y escalabilidad.# comparacion-programacion-python

\# Ejemplos del Mundo Real - Programación Orientada a Objetos



\## Descripción del Caso

Este proyecto representa un sistema básico de gestión de una biblioteca utilizando Programación Orientada a Objetos (POO) en Python.



\## Contexto del Mundo Real

En una biblioteca existen libros que pueden ser prestados y devueltos por los usuarios.



\## Conceptos de POO Aplicados

\- Clases

\- Objetos

\- Atributos

\- Métodos

\- Encapsulación

\- Interacción entre objetos

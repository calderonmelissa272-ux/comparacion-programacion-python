# Ejemplo del Mundo Real usando Programación Orientada a Objetos

## Descripción del caso

Este proyecto modela un sistema de gestión de una biblioteca utilizando
Programación Orientada a Objetos (POO).

En el mundo real, una biblioteca necesita administrar libros, saber si
están disponibles y permitir que sean prestados o devueltos por los usuarios.
Este ejemplo simula ese comportamiento mediante clases y métodos en Python.

## Clases utilizadas

- Libro: representa un libro de la biblioteca, con atributos como título,
  autor y estado de disponibilidad.
- Biblioteca: administra una colección de libros y permite prestar
  y devolver libros.

## Objetivo

Demostrar el uso de los principios de la Programación Orientada a Objetos,
como encapsulación, modularidad y reutilización de código, aplicados
a una situación real.
